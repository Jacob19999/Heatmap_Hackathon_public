========================================
  Burn Equity Index — Interactive Dashboard
  Team 15 | Challenge Area 3
========================================

REQUIREMENTS
------------
- Python 3 (any recent version)
- A modern web browser (Chrome, Firefox, Edge)
- Internet connection (for map tiles and chart library)

HOW TO RUN
----------
1. Unzip dashboard.zip into any folder.

2. Open a terminal / command prompt in that folder.

3. Run:

       python serve.py

4. Your browser will open automatically to http://127.0.0.1:8080

5. Press Ctrl+C in the terminal to stop the server when done.

WINDOWS SHORTCUT
----------------
Double-click start.bat — it does steps 2-4 for you.

MAC / LINUX SHORTCUT
--------------------
Double-click start.sh (or run: bash start.sh)

TROUBLESHOOTING
---------------
- "python" not found?  Try "python3 serve.py" instead.
- Port 8080 in use?    Run "python serve.py 9000" to pick another port.
- Maps not loading?    Make sure you have an internet connection.
